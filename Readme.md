Sales Data Cleaning

Overview
This project demonstrates data cleaning and preprocessing using Python and Excel.
The raw dataset contained missing values, duplicate records, and inconsistent data. Python was used to clean and prepare the dataset for analysis.
 Objectives
- Remove duplicate records
- Handle missing values
- Standardize data
- Improve data quality
- Create a cleaned dataset

Tools Used
- Python
- Pandas
- NumPy
- VS Code
- Microsoft Excel
- GitHub
Workflow
Step 1: Raw Dataset
Original data was stored in Excel.
 Step 2: Data Cleaning
The following operations were performed:
- Removed duplicates
- Handled missing values
- Corrected data formats
- Cleaned unnecessary records
Step 3: Export Clean Data
Cleaned dataset was saved as a new Excel file.

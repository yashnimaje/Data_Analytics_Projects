# Sales Data Cleaning Project

## Overview

This project focuses on cleaning and preprocessing sales data using Python and Excel.

The dataset contained missing values, duplicate records, and formatting issues. These problems were resolved using Python libraries such as Pandas and NumPy.

---

## Objectives

- Remove duplicate records
- Handle missing values
- Correct inconsistent formatting
- Improve data quality
- Generate a clean dataset

---

## Tools Used

- Python
- Pandas
- NumPy
- VS Code
- Microsoft Excel
- GitHub

---

## Project Files

### Data Folder

- Sales_Data_Raw.xlsx
- Sales_Data_Cleaned.xlsx

### Code Folder

- sales_data_cleaning.py


## Workflow

1. Load Raw Sales Data
2. Identify Missing Values
3. Remove Duplicates
4. Clean and Standardize Data
5. Export Clean Dataset

---

## Result

The raw sales dataset was successfully cleaned and prepared for further analysis and visualization.
